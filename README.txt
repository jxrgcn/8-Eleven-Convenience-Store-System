System Overview
This project contains the front-end user interface for the Inventory and Sales Monitoring System.

System Description:
The 8-Eleven Sari-Sari Store Management System is a web-based inventory and sales management system designed to help employees efficiently manage daily store operations. It centralizes product, category, stock, supply, and transaction records while providing real-time sales summaries and inventory monitoring through an easy-to-use dashboard.

Data Identification:
The system stores information needed for daily store operations. This includes product details, product categories, stock records, supplier information, sales transactions, transaction items, and employee accounts. These data are used to monitor inventory, process sales, and generate accurate reports.

Entity Identification:
tb_users
Stores employee account information and login credentials used to access the system.

tb_categories
Contains the available product categories and is used to organize and filter products.

tb_products
Stores information about every product sold in the store. Each product is linked to a category and contains details such as its price and available stock.

tb_stocks
Records stock movements whenever products are added or updated in the inventory.

tb_supplies
Stores information about supplier deliveries, including the products received and their quantities.

tb_transactions
Keeps a record of every sales transaction processed in the store.

tb_transaction_items
Stores the individual products included in each transaction, along with their quantity and subtotal.
